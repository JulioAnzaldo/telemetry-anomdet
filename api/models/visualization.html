Visualization Utilities¶

Visualization utilities for telemetry and anomaly scores.

Provides functions for plotting time-series signals and anomaly markers.

telemetry_anomdet.visualization.plot_anomalies()[source]¶

Placeholder for future visualization functions.

Contains helper functions for plotting anomaly scores, and flagged windows. Typical visualizations include: - Anomaly score vs time - Sensor readings with anomaly overlays - Scatter plots of normal vs anomalous data
