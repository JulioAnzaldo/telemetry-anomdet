Telemetry Anomaly Detection Toolkit
===================================

Welcome to the **telemetry-anomdet** documentation!

This toolkit provides modular utilities for **preprocessing**, **feature extraction**, 
and **supervised/unsupervised anomaly detection** in live telemetry streams.
